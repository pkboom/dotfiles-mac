# This plugin adds `clearestate` shell commands.

## Installation

```sh
git clone https://github.com/pkboom/clearestate.git $ZSH_CUSTOM/plugins/clearestate
```

Add the plugin in `~/.zshrc`

```sh
plugins=(... clearestate)
```

Update the shell

```
source ~/.zshrc
```

## Usage

```sh
$ce<space><tab>
```
